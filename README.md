# Emergency Identification System

Sistema de identificação de emergência acessível via QR Code,
projetado para crianças autistas com foco em:

- Acessibilidade
- Segurança
- Offline-first (PWA)
- Privacidade por padrão

## Stack
HTML + CSS + JS + PWA

## Executar
Abrir index.html ou deploy no GitHub Pages.

## Segurança
- Sem backend
- Sem coleta de dados
- Sem analytics
- Dados mínimos necessários

## Roadmap
- Modo crise
- Multi idioma
- Customização por família
